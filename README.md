## Projet API (Groupe 31: Clément LE BAS)

##### Commande lancement de l'application :

* Serveur: mvnw spring-boot:run
* Navigateur: http://localhost:4000/api

Si le front ne s'affiche pas:
* Installer Node.js
* cmd: 
  * npm install 
  * npm run build
